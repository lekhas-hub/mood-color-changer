# Mood Color Changer

## Description
This project changes the background color and message based on the selected mood.

## Features
- Four mood buttons
- Dynamic background color change
- Mood message display

## Concepts Used
- HTML
- CSS
- JavaScript
- DOM manipulation

## How to Run
Open the project in browser and click on any mood button.
